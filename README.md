# Домашнее задание по Python.

## Структура репозитория

```
.
├── ip_validator         # директория проекта
├── Screenshots          # директория со скриншотами рабочей джобы в Jenkins
├── Tests                # директория для фалйлов тестирования
├── Jenkinsfile          # файл настройки pipeline в Jenkins
├── report.txt           # Краткий отчет о найденных ошибках
├── requirements.txt     # файл с python зависимостями
├── setup.py             # для использования репозитория как python пакета
```
